allocStore :: DC m => Name -> Definitional m Value Source #

Creates a location for a given name. This is particular for the GRIN store structure, where the Store operation must be part of a Bind, thus there will be always a name to bind to, which should hold the address of the created location.

In this Definitional interpreter for every Store operation that the interpreter evaluates it must creates a new location.
